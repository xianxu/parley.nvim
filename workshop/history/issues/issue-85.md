## Description

oftentimes, I want to ask quick question around terminologies of a term. would love a way to quickly be able to do that.

one design would be, visual select some text, then do a <C-g>i, which convert to start a child chat with that selected text with: what is "the selected text" as first question. then in terms of presentation inside original chat, it might be displayed as [🌿:selected text](the-to-child-file). this way you sort of have a footnote effect of the subchat.

the difference between this syntax and the full line chat branch, is that here, more likely we are just trying to get a bit more context for the selected text, and thus can simplify the visual presentation from "full topic of the chat" to be just a "🌿:" and hyperlink the selected text, as the topic is the selected.
