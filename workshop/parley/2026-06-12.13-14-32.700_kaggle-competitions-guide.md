### What Kaggle Competitions Are

Kaggle is a data science platform (owned by Google) where companies and researchers post **machine learning problems**, and people compete to build the best-performing model. Each competition gives you:

- A **dataset** (training + test data)
- A **problem definition** (e.g., predict house prices, classify images)
- An **evaluation metric** (how submissions are scored, e.g., accuracy, RMSE)
- A **leaderboard** ranking all participants

You submit predictions, and Kaggle scores them automatically against a hidden test set.

### Types of Competitions

- **Featured** — real-world problems sponsored by companies, often with cash prizes
- **Research** — experimental/academic problems
- **Getting Started** — permanent beginner competitions (e.g., *Titanic*, *House Prices*) with no deadline — ideal for learning
- **Playground** — fun, low-stakes problems for practice

### How to Learn and Get Help

The Kaggle ecosystem is itself the best learning resource:

| Resource | What it's for |
|---|---|
| **Kaggle Learn** | Free micro-courses (Python, ML, pandas, deep learning) |
| **Notebooks (Kernels)** | Public code others share — read top solutions to learn techniques |
| **Discussion forums** | Ask questions, read strategy threads per competition |
| **Datasets** | 50k+ free datasets to practice on |

### Suggested Learning Path

1. Start with the **Titanic** competition — the de facto "Hello World" of ML.
2. Read **public notebooks** for that competition (sort by "Most Votes") to see how others approach it.
3. Take the relevant **Kaggle Learn** courses (Intro to ML → Intermediate ML → Pandas).
4. Make your first submission, then iterate to climb the leaderboard.
